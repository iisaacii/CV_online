Experiencia de Proyectos
Sistema de Predicción de Criptomonedas (Proyecto Personal)
Tecnologías: Python, LSTM/TCN, TensorFlow, Pandas, RSI, EMAs, ATR
Problema: Anticipar movimientos de precios en criptomonedas (DOGE/USDT) para identificar puntos de entrada con alta expectativa.
Solución: Desarrollé un sistema completo con redes neuronales LSTM/TCN para predecir retornos de criptomonedas, construyendo pipelines de datos OHLCV, generando features de series de tiempo (EMAs, RSI, volatilidad, mínimos locales) y combinando señales probabilísticas con confirmadores técnicos y gestión de riesgo basada en ATR.
Resultado: Mejoré significativamente la precisión de entradas frente a estrategias manuales, con mejor profit factor y drawdown más controlado.
Predicción de Demanda Hotelera - Fortem Capital
Tecnologías: Python, Prophet, XGBoost, C#/.NET, Java/Spring Boot, React, Power BI, SQL, MongoDB
Problema: Proyectar demanda de vivienda y ocupación hotelera en zonas turísticas para optimizar decisiones de inversión inmobiliaria.
Solución: Implementé modelos de series de tiempo (Prophet) y XGBoost Regressor para predecir demanda y precios, integrando datos históricos de ventas, ocupación y variables económicas. Desarrollé microservicios en C# y Java con Spring Boot para exponer APIs con resultados y creé dashboards dinámicos en React y Power BI.
Resultado: Reduje en más del 20% el error de predicción y proporcioné una plataforma dinámica para decisiones de inversión en tiempo real.
Clasificación de Calidad de Minerales (Proyecto Académico)
Tecnologías: Python, XGBoost Classifier, scikit-learn, Pandas
Problema: Clasificar calidad del mineral (alta, media, baja) al momento de extracción, ya que los análisis de laboratorio tardaban días en procesarse.
Solución: Desarrollé un modelo de clasificación con XGBoost utilizando variables como pH, densidad, humedad y ubicación. Realicé preprocesamiento completo: limpieza, normalización y balanceo de clases, evaluando con F1 Score para optimizar precisión.
Resultado: Logré 85% de precisión en clasificación, reduciendo tiempo de clasificación de días a minutos y mejorando significativamente la eficiencia operativa del proceso minero.
